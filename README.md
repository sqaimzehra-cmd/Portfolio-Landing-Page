# Syeda Qaim Zehra Portfolio

This project is a senior designer portfolio landing page for Syeda Qaim Zehra, built with HTML, CSS, and JavaScript. It presents a more professional and visually refined impression than a basic classroom demo.

## Features

- Personal portfolio hero section with strong typography and branding
- Responsive layout for desktop and mobile screens
- Animated metric counters using JavaScript
- Interactive creative focus text
- Clickable design principle cards with live content updates

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Files

- `index.html`
- `style.css`
- `script.js`

## Purpose

This project demonstrates:

- clean page structure with HTML
- custom styling and responsive design with CSS
- user interaction and DOM updates with JavaScript
- a polished presentation suitable for a portfolio landing page
